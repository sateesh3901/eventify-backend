## ✨ Features

- 🔐 **JWT Authentication** — Secure login, register, logout
- 👥 **Role-Based Access Control** — User, Host, Admin roles
- 🎪 **Event Management** — Full CRUD with filters and search
- 🎟️ **QR Code Ticketing** — Auto-generated unique QR codes
- 💳 **Razorpay Payments** — Secure payment integration
- 💼 **Career Fair Module** — Job openings & resume applications
- 📊 **Real-time Stats** — Dashboard analytics for all roles
- 📷 **QR Scanner** — Ticket validation at event entry
